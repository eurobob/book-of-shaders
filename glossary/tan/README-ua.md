## tan
Повертає тангенс переданого кута

### Оголошення
```glsl
float tan(float angle)  
vec2 tan(vec2 angle)  
vec3 tan(vec3 angle)  
vec4 tan(vec4 angle)
```

### Параметри
**```angle```** — величина кута в радіанах, тангенс якого потрібно повернути.

### Опис
**```tan()```** повертає тригонометричний тангенс кута.
