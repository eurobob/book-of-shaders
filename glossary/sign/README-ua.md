## sign
Виділяє знак параметра

### Оголошення
```glsl
float sign(float x)  
vec2 sign(vec2 x)  
vec3 sign(vec3 x)  
vec4 sign(vec4 x)
```

### Параметри
**```x```** — значення, з якого слід витягти знак.

### Опис
**```sign()```** повертає: -1.0, якщо x менше 0.0; 0.0, якщо x дорівнює 0.0; +1.0, якщо x більше 0.0.
