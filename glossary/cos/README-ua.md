## cos
Повертає косинус параметра

### Оголошення
```glsl
float cos(float angle)  
vec2 cos(vec2 angle)  
vec3 cos(vec3 angle)  
vec4 cos(vec4 angle)
```

### Параметри
**```angle```** — величина в радіанах, косинус якої потрібно повернути.

### Опис
**```cos()```** повертає тригонометричний косинус кута.
