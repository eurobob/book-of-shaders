## length
Обчислює довжину вектора

### Оголошення
```glsl
float length(float x)  
float length(vec2 x)  
float length(vec3 x)  
float length(vec4 x)
```

### Параметри
**```x```** — вектор, довжину якого потрібно обчислити.

### Опис
**```length()```** повертає довжину вектора.
