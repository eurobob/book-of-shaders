## ceil
Знаходить і повертає найближче ціле число, яке більше або дорівнює параметру

### Оголошення
```glsl
float ceil(float x)  
vec2 ceil(vec2 x)  
vec3 ceil(vec3 x)  
vec4 ceil(vec4 x)
```

### Параметри
**```x```** — значення для обробки.

### Опис
**```ceil()```** повертає значення, що дорівнює найближчому цілому числу, яке більше або дорівнює **`x`**.
