## not
Логічно інвертує булевий вектор

### Оголошення
```glsl
bvec2 not(bvec2 x)  
bvec3 not(bvec3 x)  
bvec4 not(bvec4 x)
```

### Параметри
**```x```** — вектор, який потрібно інвертувати.

### Опис
**```not()```** логічно інвертує булів вектор **`x`**. Повертає новий булів вектор, для якого кожен елемент **`i`** обчислено як **`!x[i]`**.
