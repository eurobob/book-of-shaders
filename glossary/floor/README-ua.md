## floor
Знаходить і повертає найближче ціле число, яке менше або дорівнює параметру

### Оголошення
```glsl
float floor(float x)  
vec2 floor(vec2 x)  
vec3 floor(vec3 x)  
vec4 floor(vec4 x)
```

### Параметри
**```x```** — значення для обробки.

### Опис
**```floor()```** повертає значення, що дорівнює найближчому цілому числу, яке менше або дорівнює **`x`**.
