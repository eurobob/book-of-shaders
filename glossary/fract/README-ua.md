## fract
Повертає дробову частину аргументу

### Оголошення
```glsl
float fract(float x)  
vec2 fract(vec2 x)  
vec3 fract(vec3 x)  
vec4 fract(vec4 x)
```

### Параметри
**```x```** — значення для обробки.

### Опис
**```fract()```** повертає дробову частину від **`x`**. Це обчислюється як **```x - floor(x)```**.
