## int
Цілочисельний тип даних

### Приклад
```glsl
int aInt = 42;
int bInt = int(aBool);
int cInt = int(aFloat);
```

### Опис
**```int```** — використовується для позначення змінних із цілочисельними значеннями. Також є одноіменна функція для приведення даних до відповідного типу.
