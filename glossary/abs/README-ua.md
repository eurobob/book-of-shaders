## abs
Повертає абсолютне значення параметра.

### Оголошення
```glsl
float abs(float x)  
vec2 abs(vec2 x)  
vec3 abs(vec3 x)  
vec4 abs(vec4 x)
```

### Параметри
**```x```** — значення, яке потрібно повернути в абсолют.

### Опис
**```abs()```** повертає абсолютне значення **`x`**.
