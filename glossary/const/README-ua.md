## const
Константний кваліфікатор

### Приклад
```glsl
const float PI = 3.14159265359;
```

### Опис
**```const```** — кваліфікатор, який застосується перед оголошенням будь-якої змінної, щоб вказати, що значення цієї змінної не буде і не може бути змінено.
