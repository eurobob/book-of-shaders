## asin
Повертає арксинус параметра

### Оголошення
```glsl
float asin(float x)  
vec2 asin(vec2 x)  
vec3 asin(vec3 x)  
vec4 asin(vec4 x)
```

### Параметри
**```x```** — значення, арксинус якого потрібно повернути.

### Опис
```asin()``` повертає кут, тригонометричний синус якого дорівнює **`x`**
