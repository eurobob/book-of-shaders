## exp2
Повертає число 2 піднесене до заданого степеня.

### Оголошення
```glsl
float exp2(float x)  
vec2 exp2(vec2 x)  
vec3 exp2(vec3 x)  
vec4 exp2(vec4 x)
```

### Параметри
**```x```** — значення степеня, до якого буде піднесено число 2.

### Опис
**```exp2()```** повертає число 2 у степені **`x`**.
