## acos
Повертає арккосинус параметра

### Оголошення
```glsl
float acos(float x)  
vec2 acos(vec2 x)  
vec3 acos(vec3 x)  
vec4 acos(vec4 x)
```

### Параметри
**```x```** — значення, арккосинус якого потрібно повернути.

### Опис
**```acos()```** повертає кут, тригонометричний косинус якого дорівнює **`x`**.
