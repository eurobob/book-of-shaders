## sqrt
Повертає значення квадратного кореня

### Оголошення
```glsl
float sqrt(float x)  
vec2 sqrt(vec2 x)  
vec3 sqrt(vec3 x)  
vec4 sqrt(vec4 x)
```

### Параметри
**```x```** — значення, з якого потрібно взяти квадратний корінь.

### Опис
**```sqrt()```** повертає квадратний корінь з **`x`**.
